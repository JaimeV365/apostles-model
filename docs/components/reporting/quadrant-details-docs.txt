# QuadrantDetails Component

## Overview
The QuadrantDetails component displays detailed analysis for specific quadrants in the Apostles Model, including statistics, trends, and insights. It uses mini BarCharts for compact visualizations.

## Features
- Detailed quadrant statistics
- Mini distribution charts
- Key findings display
- Group insights
- Premium analysis features

## Interface

### Props
```typescript
interface QuadrantDetailsProps {
  quadrant: QuadrantType;
  stats: QuadrantStatistics;
  isPremium: boolean;
  satisfactionScale: string;
  loyaltyScale: string;
}

type QuadrantType = 'loyalists' | 'hostages' | 'mercenaries' | 'defectors';
```

## Usage

### Basic Usage
```tsx
<QuadrantDetails
  quadrant="loyalists"
  stats={quadrantStats}
  isPremium={false}
  satisfactionScale="1-5"
  loyaltyScale="1-5"
/>
```
